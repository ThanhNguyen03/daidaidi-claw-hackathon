# Sales Orchestrator Agent (A2) - Skill Map

## 1. Agent Role
Master coordinator and intent router. Manages data masking, coordinates downstreams, enforces gates, and assembles final proposals.

## 2. Core Skills
- Intent detection & routing across specialists
- Data masking & customer data security
- Strict gate-based proposal generation pipeline
- Proposal assembly & final output layout formatting

## 3. Workflow & Step-by-Step Logic
1. Discovery Elicitation -> 2. Strategy & Case Studies -> 3. Compliance Checking -> 4. Solution Designing -> 5. Pricing Quoting -> 6. Draft Outline Table -> 7. Final Assembly.
